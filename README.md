# 🛡️ SilentGuard

> **Help Begins Before You Ask.**

SilentGuard is an AI-powered personal safety platform that proactively detects potential emergencies using smartphone sensor data. Instead of waiting for users to manually trigger an SOS, SilentGuard verifies their safety and automatically initiates emergency assistance when needed.

---

## 🚨 The Problem

Traditional emergency systems rely on users to manually request help. During accidents, medical emergencies, or assaults, victims may be unable to unlock their phone or press an SOS button, delaying critical assistance.

---

## 💡 Our Solution

SilentGuard transforms a smartphone into an intelligent safety companion by continuously monitoring sensor activity and identifying potential emergencies.

When an abnormal event is detected, the system:

1. Detects unusual movement or impact.
2. Starts an intelligent verification countdown.
3. Cancels the alert if the user confirms they are safe.
4. Automatically sends an SOS with live location if there is no response.
